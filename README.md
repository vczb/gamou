# Pedido (Delivery) platform

## Stack

- [TypeScript](https://www.typescriptlang.org/)
- [NextJS](https://nextjs.org/)
- [Tailwind CSS](https://tailwindcss.com/)

## Envirovment variables

```
NEXT_PUBLIC_LS_KEY=my-ls-key
```

## Install & run

```bash
$ yarn install
```

```bash
$ yarn dev
```
